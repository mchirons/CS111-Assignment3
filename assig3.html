Problem

The file WordByWord.java consists of a single class with a main method and several other methods.  We have replaced each of the bodies of these other methods with a line that says

      return "" ;// replace this line

Each method has a comment right before it which describes what it should return. Your job is to replace each of these "replace this line" lines with as much code as you need to make the method return what it should.

Your main method will not be graded.  We suggest you put code in it to call your methods with test data and to print out the results.  Be sure that each of the other methods works - we will be testing them all.

You may assume that no erroneous values will be given as arguments to your methods.  You may assume that all parameters to your methods will

  • contain only letters and spaces

  • will not have spaces before the first word or after the last

  • will  have exactly one space between each pair of adjacent words

You should be careful that the same is true of the result returned by each of your methods.  You should not  change the case of any letters. 

Hint: it will probably be useful to have the code for some of the methods use (call) others of your methods.

You will not need the IO module since none of the modules we test will do input or output.  Instead they will get their data as method arguments and give their answers by returning values.  You may use the IO module's methods in your main method to output test results. IO.outputStringAnswer, in particular, makes it clear where you have extra leading or trailing spaces.

Some rules to follow

You must replace the lines marked "// replace this line" with code that makes each method work.

You may change the main method as you wish as long as the file still compiles without error messages in a folder containing only this file and IO.java.

You may not change anything else in the file. For instance, you may not add import statements. If you are working in Eclipse, beware that Eclipse may add a package statement at the top of the file. If so, be sure to delete it from the file you submit.
